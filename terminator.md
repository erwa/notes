### Hotkeys

Ctrl + Shift + X - maximize current active window
Ctrl + Shift + E - split vertically
Ctrl + Shift + O - split horizontally
Alt + <arrow> - shift focus
Ctrl + Page Up/Down - change tab
Ctrl + Shift + T - new tab
