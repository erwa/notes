# Inserting and Deleting Rows

### Insert a row above
```
Alt + I, R

# Mac
Ctrl + option + I, R
```

### Insert a row below
```
Alt + I, W

# Mac
Ctrl + option + I, W
```

### Delete a row
```
Alt + E, D

# Mac
Ctrl + option + E, D
```

### Not possible to format (e.g.: bold) part of a cell
It's all or nothing. See https://productforums.google.com/forum/#!topic/docs/PELcAqnlSoE%5B1-25-false%5D.

### Get current row
```
=ROW()
```
