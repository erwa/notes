### Debugging

Add a Run Configuration.

To execute gdb commands, in Debug Console, issue commands using `-exec COMMAND`, e.g. `-exec print MyFunction()`.


### Set C++ standard to use

### Keyboard shortcuts

```
Ctrl+J: show/hide terminal
Ctrl+`: switch between editor and terminal
Ctrl+Shift+space: show documentation (Javadoc, etc.)
```
